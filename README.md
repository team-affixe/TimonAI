# TimonAI

Ein selbst programmiertes User-Interface für Ollama, das eine einfache, moderne Oberfläche für die lokale Nutzung von Sprachmodellen bietet.

## Features

- Intuitive Benutzeroberfläche für Ollama
- Lokale Modell-Integration
- Schnelle Eingabe- und Ausgabeanzeige
- Übersichtlich gestaltetes UI-Layout
- Leicht erweiterbar und anpassbar

## Installation

1. Repository klonen oder Projektordner öffnen.
2. Stelle sicher, dass Ollama auf deinem Rechner installiert und konfiguriert ist.
3. Öffne die HTML-Datei im Browser oder starte einen lokalen Webserver.

## Nutzung

1. Starte Ollama lokal.
2. Öffne das UI im Browser.
3. Gib deine Anfrage ein.
4. Sende die Anfrage und erhalte die Antwort vom Modell.
